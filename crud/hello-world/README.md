# Hello CRUD!

## Goals
- create simple CRUD "app" with minimum dependencies
  - e.g. just have Node, Express, and Mongo
- implement each CRUD method
  - Create (POST)
  - Read (GET)
  - Update (PUT)
  - Delete (DELETE)
- run on localhost
- have fun!
